## Features

* Load both binary (Sound files, images) and text files (JSON, XML, TEXT, CSS, JAVASCRIPT, SVG, XML)
* XHR2 ready
* Lightweight (2KB) =)
* Preloading
* Lazy Loading
* Queue Loading
